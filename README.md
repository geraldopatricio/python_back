## Gerando a tabela
```bash
CREATE TABLE `tb_produto` (
	`id` INT(11) NOT NULL AUTO_INCREMENT,
	`codigo` INT(11) NOT NULL,
	`produto` VARCHAR(100) NOT NULL COLLATE 'utf8mb4_general_ci',
	PRIMARY KEY (`id`) USING BTREE
);
    
```
